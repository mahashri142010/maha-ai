# Maha AI 🤖

A modern AI chatbot built with Python, Flask, and Llama 3.3.

## Features
- 💬 Real-time AI chat
- 🧠 Remembers conversation history
- 🎨 Beautiful glassmorphism UI
- 🗑️ Clear chat button
- ⚡ Powered by Llama 3.3 70B

## Tech Stack
- Python + Flask
- Groq API (Llama 3.3)
- HTML + CSS + JavaScript
